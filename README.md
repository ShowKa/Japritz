# Japritz

Webページのコンテンツを速読するためのChromeプラグインです。

## 概要
日本語を分節区切りして、次々と表示します。
chrome プラグインです。

## 使い方
1. chromeにJapritzをインストールします。
1. Webページ上で文章を選択。
1. はさみのアイコンをクリック。
